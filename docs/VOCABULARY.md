# Словарь домена

Термины бизнес-логики приложения для обучения чтению (слоги, выбор по голосу).

## Задания и раунды

| Термин                    | Описание                                                                                                                                         |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Задание (Task)**        | Тип упражнения. Сейчас: «выбери слог», «собери слог» (drag-and-drop букв в слоты). Позже: другие типы и микс заданий в одной сессии.             |
| **Конфигурация задания**  | Параметры конкретного типа: сложность, количество вариантов, источник слогов и т.д. Задания предполагаются модульными и настраиваемыми.          |
| **Раунд**                 | Один вопрос в рамках задания: набор вариантов + целевой объект (что нужно выбрать).                                                              |
| **Слог**                  | Единица для задания «выбери слог» — пара букв (или один символ), напр. «НА», «КУ».                                                               |
| **Целевой слог (target)** | Слог, который диктор просит выбрать в этом раунде.                                                                                               |
| **Варианты (options)**    | Список слогов на экране. После неправильного выбора выбранный вариант убирается, остальные остаются.                                             |
| **Попытка**               | Один клик по варианту. Правильная — раунд завершён; неправильная — озвучка выбранного, подсказка (опционально), удаление варианта.               |
| **Сессия**                | Серия раундов подряд. Позже может объединять разные типы заданий.                                                                                |
| **Собери слог**           | Задание: голос говорит «Собери слог XY», буквы разбросаны по экрану, в центре полупрозрачные слоты; нужно перетащить буквы в правильном порядке. |
| **Слот**                  | Место для одной буквы в задании «собери слог». Два слота = слог из двух букв.                                                                    |

## Сложность и контент

| Термин              | Описание                                                                                            |
| ------------------- | --------------------------------------------------------------------------------------------------- |
| **Сложность**       | Уровень раунда. Сейчас: число вариантов на экране (3–6). Позже: тип слогов, длина слов и т.д.       |
| **Источник слогов** | Откуда берутся слоги для раунда. Пока — фиксированный список; позже возможна генерация по правилам. |

## Озвучка и подсказки

| Термин                   | Описание                                                                                                                              |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Озвучка (TTS)**        | Произнесение фразы голосом. Провайдер — браузерный speechSynthesis или внешний API; в коде используется единый программный интерфейс. |
| **Подсказка при ошибке** | После неправильного выбора: озвучить выбранный слог, опционально объяснение («НА — это Н и А»), затем вариант исчезает.               |

| Термин        | Описание                                                                                                                              |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
