# Задание «Прочитай слово и выбери картинку»

На экране показывается слово (визуально разбитое на слоги) и несколько вариантов картинок. Нужно прочитать слово и выбрать картинку, которая ему соответствует.

## Конфигурация и раунд

- **Тип задания:** `readWordPicture`.
- **Раунд:** слово (`word`), список вариантов картинок (`options`: `{ id, alt }[]`), id правильного варианта (`correctId`).
- Раунды создаются в `rounds.ts` из фиксированного набора слов; картинки пока только с `alt`, `src` подставляется позже.

## Поведение

- После нажатия «Начать» озвучивается инструкция «Прочитай слово и выбери картинку».
- Слово отображается по слогам: согласная+гласная — один визуальный блок (стили как в задании «Сложи слог»), последовательные согласные — отдельно. По нажатию на слог озвучивается только этот слог (TTS).
- Выбор правильной картинки → озвучка «Правильно» → следующий раунд. Неправильная → озвучка фидбека, выбранный вариант исчезает.

## Код

- Типы: `types.ts`.
- Фабрика раундов: `rounds.ts`.
- Состояние и экшены: `readWordPictureSlice.ts`.
- Озвучка: `readWordPictureSaga.ts`.
- UI: `ReadWordPictureRoundView.tsx`, `ReadWordPictureRoundContainer.tsx`.
- Разбиение слова на части: `@/domain/wordSyllables.ts` (`splitWordIntoParts`).
