# Сценарии тестов: «Выбери слог»

Описание в терминах предметной области и элементов интерфейса. В тестах привязка к экрану — по кнопке «Начать», по группе вариантов слогов и по кнопке с конкретным слогом; озвучку проверяем через мок TTS.

**Элементы интерфейса:** кнопка «Начать», варианты слогов (кнопки с буквосочетаниями), подсказка «Слушай задание…».

## Сценарий 1: Успешный путь

1. На экране раунд «Выбери слог НА»: видна только кнопка «Начать».
2. Пользователь нажимает «Начать».
3. Озвучивается инструкция «Выбери слог» и целевой слог; на экране появляются варианты слогов, среди них — правильный слог (НА).
4. Пользователь нажимает на вариант с правильным слогом.
5. Озвучивается «Правильно», задание переходит к следующему раунду.

## Сценарий 2: Ошибка и повтор

1. На экране раунд «Выбери слог НА», пользователь нажимает «Начать»; появляются варианты слогов.
2. Пользователь нажимает на неправильный вариант (например «НО»).
3. Озвучивается подсказка по выбранному слогу; этот вариант исчезает с экрана, остальные кнопки слогов остаются.
4. Пользователь нажимает на вариант с правильным слогом (НА).
5. Озвучивается «Правильно».

---

**Привязка в тестах (data-testid):** кнопка «Начать» — `pick-syllable-start`, группа вариантов — `pick-syllable-options`, кнопка слога — `pick-syllable-option-{слог}`.
