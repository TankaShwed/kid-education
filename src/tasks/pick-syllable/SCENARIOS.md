# Сценарии тестов: «Выбери слог»

Описание в терминах предметной области и элементов интерфейса. В тестах привязка к экрану — по кнопке «Начать», по группе вариантов слогов и по кнопке с конкретным слогом; озвучку проверяем через мок TTS.

**Элементы интерфейса:** кнопка «Начать», варианты слогов (кнопки с буквосочетаниями), подсказка «Слушай задание…».

## Сценарий авто теста

Пусть задание сконфигурированно:

```typescriptreact
const defaultRound = {
  type: 'pickSyllable' as const,
  target: 'НА',
  options: ['НО', 'НА', 'КА', 'КУ'] as const,
};
```

- Пользователь видит кнопку начать и нажимает на неё.
- Произносится "выбери слог НА" и на экране появляются кнопки со слогами 'НО', 'НА', 'КА', 'КУ'
- При нажатии на любой кроме НА произносится. Неправильно выбери слог НА.
- При клике на НА произносится "Правильно это НА" и формируется следующий раунд этого же задание с другими настройками
