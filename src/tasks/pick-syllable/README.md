# Задание «Выбери слог»

Голос называет слог, ребёнок выбирает его из нескольких вариантов на экране.

## Назначение

Задание на узнавание слога на слух: озвучивается целевой слог, пользователь нажимает на один из предложенных вариантов. При ошибке выбранный вариант исчезает, озвучивается короткий фидбек; при правильном ответе — «Правильно» и переход к следующему раунду.

## Сценарий раунда

1. Показывается кнопка «Начать» и варианты слогов (пока без озвучки).
2. Пользователь нажимает «Начать» → озвучиваются фраза «Выбери слог» и целевой слог.
3. Пользователь выбирает вариант. Если неверно — вариант убирается с экрана, озвучивается фидбек, после чего снова можно выбирать. Если верно — озвучивается «Правильно», затем загружается следующий раунд.

## Конфигурация и раунд

Сложность задаётся числом вариантов на экране (3–6), настраивается в UI выбора сложности (только для этого задания). Раунд создаётся фабрикой из домена слогов.

**Типы и фабрика:** см. `src/domain/types.ts` (PickSyllableRound, PickSyllableTaskConfig), `src/domain/rounds.ts` (createPickSyllableRound).

## Состояние и события

В store под ключом `pickSyllable` хранятся варианты на экране (после ошибок список уменьшается), статус (ожидание / ошибка / правильный ответ), флаги начала раунда и озвучки инструкции. События: начало раунда, выбор варианта (верный/неверный); сага реагирует на них озвучкой и сменой раунда.

**Подробно:** экшены и поля состояния — в `pickSyllableSlice.ts` (TSDoc).

## Сага

Озвучивает инструкцию по экшену startRound, фидбек при chooseWrong, «Правильно» и вызывает следующий раунд при chooseCorrect.

**Детали:** `pickSyllableSaga.ts` (TSDoc).

## UI

- **PickSyllableRoundView** — презентационный компонент: кнопка «Начать», варианты слогов, колбэки onStart и onChoose. Пропсы описаны в TSDoc в файле.
- **PickSyllableRoundContainer** — подключает View к store и диспатчит экшены slice. Озвучка выполняется сагой.

См. `PickSyllableRoundView.tsx`, `PickSyllableRoundContainer.tsx`.

## Подключение в приложение

- Типы и фабрика раунда: `src/domain/types.ts`, `src/domain/rounds.ts`.
- Store: reducer `pickSyllable`, в `createRound()` и `nextRound()` ветка для `pickSyllable` — `src/store/store.ts`.
- Сага: `fork(pickSyllableSaga, context)` в `src/store/sagas/rootSaga.ts`.
- App: переключение задания и ветка `currentRound.type === 'pickSyllable'` с рендером `<PickSyllableRoundContainer key={roundKey} />` — `src/App.tsx`.
